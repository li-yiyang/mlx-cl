* About
MLX-CL/IO/PNG add support for reading and writing =mlx-array= from and to PNG file
via [[https://github.com/lisp-mirror/pngload][pngload]] and [[https://www.quicklisp.org/beta/UNOFFICIAL/docs/zpng/doc/index.html][zpng]].

* Usage
** Loading
#+begin_src lisp
  (ql:quickload :mlx-cl/io/png)
#+end_src

*NOTE*: if you load [[../../image/][=mlx-cl/image=]], it would automatically load
=mlx-cl/io/png= and other image file IO support.

** Reading PNG file
#+begin_src lisp
  (mlx-array "xxx.png")
#+end_src

** Writing PNG file
#+header: :results file
#+header: :var width=200 height=200
#+begin_src lisp
  (destructuring-bind (x y)
      (mlx:meshgrid (linspace 0 1 width)
                    (linspace 0 1 height))
    (flet ((mask (ch) (where (<= (+ x y) 1) ch 0)))
      (->* (stack (mask x)                ; R
                  (mask y)                ; G
                  (mask (- 1 x y))        ; B
                  :axis 2)                ; on channel axis
        (as-dtype (* * 255) :uint8)
        (save * :output "./res/img/write-png-example.png"))))
#+end_src
